# 🏎️ Гайд: Сборка Neuro-Explorer из LEGO 51515

![Neuro-Explorer 51515 Wheels Edition](file:///C:/Users/Beknur/.gemini/antigravity/brain/decc3379-d443-4eff-bbfe-a5636f3965a7/neuro_explorer_51515_wheels_blueprint_1777634286652.png)

Этот мануал написан специально для владельцев набора **LEGO Mindstorms 51515 (Robot Inventor)**. Мы используем только те детали, которые есть в твоём наборе. 

---

## 🏗️ ШАГ 1: Скоростное Шасси (Конфигурация пользователя)

1.  **Привод**: Возьми два мотора и закрепи их по бокам большой белой рамки.
2.  **Колеса**: Надень шины на бирюзовые диски.
3.  **Порты**: 
    *   Правый мотор -> **Порт A**
    *   Левый мотор -> **Порт C**
4.  **Баланс**: Сзади прикрепи две изогнутые балки-лыжи. Они отлично скользят по полу.

## 📱 ШАГ 2: Neural Cradle (Трон для телефона)
Телефон должен сидеть мертво и смотреть вперед.

1.  **Стойка**: Построй вертикальную рамку из белых балок.
2.  **Кивок (Tilt)**: 
    *   Закрепи рамку на **моторе D**.
    *   Мотор наклона -> **Порт D**.
3.  **Сенсоры**: Ультразвуковой датчик (глазки) подключи к **Порту B**.

## 🔋 ШАГ 3: Установка Хаба
1.  Зеленый Хаб 51515 закрепи горизонтально в центре базы.
2.  Убедись, что экран хаба доступен.
3.  Провода аккуратно закрепи клипсами.

## ⚡ ШАГ 4: Запуск Интеллекта
1.  Включи хаб, запусти сервер на ноуте (`npm run dev:server`).
2.  Открой Phone Page на смартфоне и вставь его в "Трон".
3.  В чате дай команду:
    > "Система, активируй режим Исследователя 51515. Правый мотор - A, Левый - C, Голова - D. Начни патруль."
