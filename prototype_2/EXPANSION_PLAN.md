## Фаза 3: Экосистема MCP (Июль — Сентябрь)

### Библиотека MCP инструментов
- [ ] Встроенные инструменты:
  - `move_forward`, `move_backward`, `turn_left`, `turn_right`, `stop`
  - `capture_image`, `detect_color`, `detect_object`, `read_text` (OCR)
  - `speak`, `listen`, `recognize_speech`
  - `read_sensors` (акселерометр, гироскоп, магнитометр)
  - `display_text`, `display_emoji`, `play_sound`, `vibrate`
  - `wait`, `wait_for_color`, `wait_for_sound`, `wait_for_touch`
- [ ] Инструменты внешних API:
  - Weather API (робот реагирует на погоду)
  - News API (робот читает заголовки новостей)
  - Умный дом (Philips Hue, Home Assistant, Tuya)
  - Spotify (робот танцует под музыку)
  - Telegram/Discord (робот отправляет сообщения)

### Конструктор кастомных MCP инструментов
- [ ] Визуальный конструктор — drag & drop для создания новых инструментов
- [ ] AI-помощник создания — «Хочу чтобы робот проверял цену биткоина» → AI создаёт MCP инструмент
- [ ] Маркетплейс инструментов — шеринг и поиск инструментов от сообщества
- [ ] Песочница для тестирования — тестирование инструментов без физического робота

### Архитектура MCP сервера
```
BrainBricks MCP Server
├── /tools/robot      — Управление моторами и движением
├── /tools/vision     — Камера, распознавание объектов, трекинг цветов
├── /tools/audio      — Речь, распознавание звуков, музыка
├── /tools/sensors    — Акселерометр, гироскоп, приближение
├── /tools/external   — Погода, умный дом, соцсети
└── /tools/custom     — Пользовательские инструменты
```
